# VIX Home Credit - End to End Analysis

## Deskripsi Singkat

Home Credit saat ini sedang menggunakan berbagai macam metode statistik dan Machine Learning untuk membuat prediksi skor kredit. Sekarang, kami meminta anda untuk membuka potensi maksimal dari data kami. Dengan melakukannya, kita dapat memastikan pelanggan yang mampu melakukan pelunasan tidak ditolak ketika melakukan pengajuan pinjaman, dan pinjaman datap diberikan dengan principal, maturity, dan repayment calendar yang akan memotivsi pelanggan untuk sukses.

## Struktur Folder/Files

- `models` : Folder yang berisi hasil save model akhir dari seluruh proses yang telah dilakukan sebelumnya.
- `vid_dashboard` : Folder yang berisi video dashboard yang telah dibuat dan dijalankan pada PowerBI.
- `HC-MergingDataProcess.ipynb` : Notebook alur proses penggabungan beberapa data utama
- `HC-AnalysisData.ipynb` : Notebook proses analisis dan visualisasi data
- `HC-ModellingProcess.ipynb` : Notebook yang berisi alur proses pemodelan dengan Machine Learning
